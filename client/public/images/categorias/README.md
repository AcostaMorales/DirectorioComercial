# Imágenes de Categorías

Esta carpeta debe contener las imágenes para las categorías de negocios.

## Imágenes Requeridas:

1. **abarrotes.jpg** - Imagen representativa de tiendas de abarrotes
2. **bares.jpg** - Imagen representativa de bares y cantinas
3. **clinicas.jpg** - Imagen representativa de clínicas médicas
4. **restaurantes.jpg** - Imagen representativa de restaurantes
5. **ferreterias.jpg** - Imagen representativa de ferreterías
6. **farmacias.jpg** - Imagen representativa de farmacias
7. **panaderias.jpg** - Imagen representativa de panaderías
8. **talleres.jpg** - Imagen representativa de talleres mecánicos

## Especificaciones:

- **Formato:** JPG o PNG
- **Tamaño recomendado:** 200x200 píxeles mínimo
- **Relación de aspecto:** Cuadrado (1:1) preferible
- **Peso:** Máximo 100KB por imagen para optimizar carga

## Nota:

Si alguna imagen no se encuentra, el sistema mostrará automáticamente un emoji de respaldo correspondiente a cada categoría.

## Fuentes Sugeridas:

- Unsplash.com (imágenes gratuitas)
- Pexels.com (imágenes gratuitas)
- Freepik.com (con licencia)
- Fotografías propias de negocios locales
